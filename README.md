## Autorzy: Justyna Ziemichód, Jakub Jaśków (G2)

## Instrukcja:

Aby poniższe instrukcje zadziałały muszą one zostać wykonane z wewnątrz folderu `src/`.

1. Start
```
docker compose up -d
```
2. Stop
```
docker compose down
```
3. Twardy reset (usunięcie wolumenów / bazy danych)
```
docker compose down -v
```
